# 🔐 Laravel Authentication & User Management System

A robust and secure authentication system built with Laravel, designed with modern best practices and extensibility in mind.

## ✨ Features

### 🔑 Authentication

- Login & Register system
- Google OAuth integration
- Two-Factor Authentication (2FA) via Email OTP

### 📧 Email Security

- Password reset via email
- Change password with OTP verification

### 🛡️ Security & Sessions

- Session management (track & control active sessions)
- Enhanced account protection with OTP verification

### 👥 Role & Permission Management

- Role-Based Access Control (RBAC) using Spatie Laravel Permission
- Flexible role and permission assignment

### 🔔 Real-time Notifications

- Real-time notification system using Laravel Reverb (WebSocket)
- Live updates without page reload
- Browser (Chrome) push notifications support
- Toggle notification preference via UI (localStorage-based)

### 📊 Activity Logging

- User activity tracking with Spatie Laravel Activitylog
- Monitor important actions across the system

## 🧱 Tech Stack

- Laravel
- Laravel Reverb (WebSocket)
- Laravel Echo
- Laravel Socialite (Google OAuth)
- Spatie Laravel Permission
- Spatie Laravel Activitylog

## 🎯 Purpose

This project serves as a core foundation for building secure and scalable web applications, with a focus on authentication, authorization, real-time communication, and user activity tracking.

---

💡 Built for extensibility — ready to integrate features like WhatsApp API (WAHA), advanced notification channels, and more.
